Impact of Job Characteristics on Salaries in the Data Industry

Overview

This project involved analyzing the impact of job characteristics on salaries in the data industry using Power BI. The goal was to provide insights into how different factors such as job title, experience level, and employment type influence salaries.

Visualizations

Interactive dashboards were created to visualize key metrics such as average salary by country, job title, and experience level.

Key Features

  • Dataset from Kaggle: Jobs in Data
  • Dimensions: Work year, job title, job category, salary currency, employee residence, experience level, employment type, work setting, company location, company size
  • Measures: Salary, salary in USD
  • Snowflake Schema: Fact table (Salary Details) and dimension tables (Work Year, Job Title, etc.)

Technologies Used

  • Power BI for data visualization
  • Data modeling using Snowflake Schema
  • Interactive dashboards with real-time data updates
  • Calculated fields for advanced metrics

Outcome

The project provided valuable insights into the factors influencing salaries in the data industry. The interactive dashboards allowed stakeholders to explore data and make informed decisions based on the visualizations.
